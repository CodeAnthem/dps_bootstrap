# core

Shared infrastructure used by every action.

| File | Role |
|------|------|
| `formatting.sh` | Console output, steps, sections |
| `userInput.sh` | Prompts and confirmations |
| `crypto.sh` | Key generation helpers |
| `actionHelpers.sh` | Action-level utilities |
| `runtime.sh` | `NDS_RUNTIME_DIR`, install log (`/tmp/nds_install.log`) |
| `import.sh` | `nds_import_*`, `nds_bootstrap_load_libs` |
